# Support

Thank you for using **GoTestX**.

This document explains how to get help and report issues.

---

# Getting Help

Before opening an issue, please check:

1. The README documentation
2. Existing GitHub issues
3. The latest release notes

Repository:

https://github.com/entiqon/gotestx

---

# Reporting Bugs

If you encounter a bug, please open a GitHub issue and include:

* Go version (`go version`)
* Operating system
* GoTestX version (`gotestx -v`)
* The command you ran
* Full output or error message

Example:

```
gotestx -cq ./...
```

Providing this information helps us diagnose problems quickly.

---

# Feature Requests

Feature suggestions are welcome.

When proposing a feature, please explain:

* The problem it solves
* Why it improves the Go testing workflow
* Example usage

Example:

```
gotestx --summary
```

---

# Security Issues

